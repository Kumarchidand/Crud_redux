# 🛠️ MERN CRUD App with Redux

This is a full-stack **CRUD (Create, Read, Update, Delete)** application built using the **MERN stack** (MongoDB, Express, React, Node.js) and **Redux** for state management. Users can create, edit, view, and delete data with a responsive and interactive UI.

---

## 🚀 Features

- 📦 Redux state management for predictable UI updates
- 🧾 Add, View, Edit, and Delete users
- 🧭 Clean Navigation using React Router
- 🎨 Stylish UI with Bootstrap
- 🔗 Backend API with Express & MongoDB
- ☁️ Data persistence using MongoDB
- 🔐 CORS and cookie-enabled secure requests
